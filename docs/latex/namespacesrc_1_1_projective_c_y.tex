FLASK METHODS \#. 

Method to find the discontinuities in the mass of a single spherical twist on a K3 surface using the discretized Laplacian. The method computes the discretized Laplacian values at each point in the (x,y) plane, and then computes the average and standard deviation of the discretized Laplacian values. The method then highlights the points in the (x,y) plane that have a discretized Laplacian value greater than the average plus the standard deviation.
