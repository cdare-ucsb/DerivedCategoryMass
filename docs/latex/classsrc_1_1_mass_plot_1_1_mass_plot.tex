\doxysubsection{Detailed Description}
As the purpose of the library is to compute the mass of spherical twists in certain chambers of stability manifolds, a useful functionality is to plot a certain 3D slice of the graph of the mass function. 

Since the \doxylink{namespacesrc_1_1_spherical_twist}{Spherical\+Twist} class allows significant flexibiliy with its catagory parameter (which effectively dictates the stability manifold we wish to graph), the majority of the plotting functionality will look the same for each catagory (currently \textquotesingle{}P1\textquotesingle{}, \textquotesingle{}P2\textquotesingle{}, and \textquotesingle{}K3\textquotesingle{}). The only difference will be the way the x and y values are generated for the plot. Thus, this class acts as wrapper to remove that redundancy and allow for easy plotting of the mass function for a given spherical twist in a given chamber of a stability manifold.
