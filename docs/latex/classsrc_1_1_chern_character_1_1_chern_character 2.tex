Initialize a Chern Character object with the specified graded element. 

The graded element should be a list of floats or integers, representing the Chern character of a coherent sheaf or vector bundle. Since Chern characters are additive on exact sequences, this allows us to also consider Chern characters of complexes of coherent sheaves.

In order to allow for quick computations, the graded element is stored as a numpy array.
