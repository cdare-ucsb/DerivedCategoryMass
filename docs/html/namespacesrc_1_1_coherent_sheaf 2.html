-------------------------------------------------------------------------—# These objects are used to represent coherent sheaves, vector bundles, and # line bundles. Line bundles (i.e. vector bundles of rank 1) will be our # building blocks for the sake of this project, since every coherent sheaf # in projective space has a resolution by line bundles. Moreover, all #
spherical twists that we will consider can be represented as a composition # of twists around (the pushforward of) line bundles. #
