Detailed Description

Class which encodes a significant porition of the mathematical information about the Drézet-Le Potier curve embedded in the ch1/ch0, ch2/ch0 plane.

An algorithmic description of how to obtain the coordinates of exceptional vector bundles is described in

https://link.springer.com/article/10.1007/s00029-017-0352-4

Method to plot the region of the ch1/ch0, ch2/ch0 plane above the Drezet-Le Potier curve.

The plot can be displayed in the browser or returned as a JSON string, and can be in 2D or 3D. Additionally, the plot can include the walls of the chambers and the colors of the boundary and walls can be customized.

Parameters
boolplot_3d A flag to indicate whether the plot should be in 3D. Default is False.
boolreturn_json: A flag to indicate whether the plot should be returned as a JSON string. Default is False.
boolshow_walls: A flag to indicate whether the walls of the chambers should be shown. Default is False.
strboundary_color: The color of the boundary of the curve. Default is 'blue'.
strwall_color: The color of the walls of the chambers. Default is 'gray'.
Returns
: A JSON string representation of the plot if return_json is True
