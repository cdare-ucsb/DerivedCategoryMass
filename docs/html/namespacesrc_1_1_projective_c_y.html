Example of a method to plot a complex degree d hypersurface using matplotlib, and save the animation either as an HTML <video> tag or as a GIF file.

The hypersurface is given by the equation

z0^d + z1^d + ... + zn^d = 1

and is restricted to the region 0 <= z0, z1 <= pi/2. The plot is done using Matplotlib.

FLASK METHODS #.

Method to find the discontinuities in the mass of a single spherical twist on a K3 surface using the discretized Laplacian. The method computes the discretized Laplacian values at each point in the (x,y) plane, and then computes the average and standard deviation of the discretized Laplacian values. The method then highlights the points in the (x,y) plane that have a discretized Laplacian value greater than the average plus the standard deviation.

Parameters
intline_bundle_1 The line bundle of the object being twisted
intline_bundle_2 The line bundle of the object being twisted by
intdegree The degree of the K3 surface
plotA flag indicating whether the discontinuities should be plotted
strcolorscale The colorscale to be used in the plot

floatx_max The maximum x value for the plot
floatx_granularity The granularity of the x-axis
floaty_min The minimum y value for the plot
floaty_max The maximum y value for the plot
floaty_granularity The granularity of the y-axis
Exceptions

Returns
