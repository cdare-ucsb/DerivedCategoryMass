Example of a method to plot a complex degree d hypersurface using matplotlib, and save the animation either as an HTML <video> tag or as a GIF file.

The hypersurface is given by the equation

z0^d + z1^d + ... + zn^d = 1

and is restricted to the region 0 <= z0, z1 <= pi/2. The plot is done using Matplotlib.

Parameters
intdegree The degree of the hypersurface
strfilename The name of the file to save the plot to
boolto_gif A flag indicating whether the plot should be saved as a GIF file
boolto_html A flag indicating whether the plot should be saved as an HTML file
inty_granularity The granularity of the y-axis
intx_granularity The granularity of the x-axis
intnframes The number of frames in the animation
intt_interval: The time interval between frames
Exceptions
ValueErrorIf both to_gif and to_html are both simultaneously set to True

STATIC PLOTTING EXAMPLES #.

First static example for plotting a degree d complex hypersurface; specifically, this just restricts the graph of

z0^d + z1^d + ... + zn^d = 1

to the region 0 <= z0, z1 <= pi/2. The plot is done using Plotly.

Parameters
intdegree The degree of the hypersurface
strfilename The name of the file to save the plot to
boolto_html A flag indicating whether the plot should be saved to an HTML file
boolreturn_json A flag indicating whether the plot should be returned as a JSON string
inty_granularity The granularity of the y-axis
intx_granularity The granularity of the x-axis

Helper function to convert the data of a spherical twist triangle to a JSON string.

The data includes the sheaf vectors, shift vectors, and dimension vectors of the objects in the triangle. This is primarily used for the front-end visualization of the spherical twist triangle in a Flask application.

Parameters
intline_bundle_1 The line bundle of the first object in the spherical twist triangle
intline_bundle_2 The line bundle of the second object in the spherical twist triangle
intline_bundle_3: The line bundle of the third object in the spherical twist triangle
intdegree The degree of the K3 surface
Exceptions
ValueErrorIf the input data is not an integer
Returns
str A JSON string representation of the spherical twist triangle data
